# Data Source Catalog (v1 priority)

## Core sources

- OEWS: occupational employment and wage estimates.
- LAUS: labor force, employment, unemployment.
- QCEW: industry employment, wages, establishments.
- ACS 5-year: demographic and socioeconomic context.
- Population Estimates: denominators and growth context.
- CBP/ZBP: business and establishment structure.
- BEA RPP: regional price parity.
- BEA GDP: local economic scale and growth.

## Phase-two sources

- LEHD/LODES and QWI
- IPEDS and College Scorecard
- O*NET
- FCC broadband
- IRS migration
- RUCA/RUCC

## Metadata requirements

Each source snapshot must store:

- Source name and release period
- Extraction timestamp
- Schema version/hash
- QA status and notes
